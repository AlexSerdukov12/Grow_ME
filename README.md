# Grow_ME
Smart hydroponic cultivation, using Arduino with various sensors.
With a connection to the application in secure communication

Arduino Sensors:
* PH 
* Water Level
* Fans
* LDR
* Temp & Humidity
* ESP32-cam
* Esp8266 WIFI (changed it to cable)
* LED 220V with connect to REALY
* Water Pump

storage :
local disk ,
cloud:
FireBase.
