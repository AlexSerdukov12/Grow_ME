# Grow_ME
Smart hydroponic cultivation, using Arduino with various sensors.
With a connection to the application in secure communication

Arduino Sensors:
* PH 
* Water Level
* Fans
* LDR
* Temp & Humidity
* ESP32-cam
* Esp8266 WIFI
